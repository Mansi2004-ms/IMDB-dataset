 IMDB Movie Review Sentiment Analyzer
The Movie Review Sentiment Analysis App is a Machine Learning web application that analyzes the sentiment of movie reviews and classifies them as Positive 😊 or Negative 😞. The application uses Natural Language Processing (NLP) techniques to preprocess text and a trained machine learning model to predict the sentiment of user-entered reviews in real time.

Built with Python, Scikit-learn, and Streamlit, this project demonstrates the complete NLP workflow, from text preprocessing and feature extraction to model training and deployment.
